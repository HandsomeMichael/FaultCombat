Fault Combat : Dodgeroll & Calamity

This mod plans to add new combat mechanics with calamity mod and other major content mods if possible.
Currently it only adds Dodgeroll with stamina

Vanilla and modded accesories now have extended abilities to the new mechanic
This mod have a lot of options / config, so be sure to check that out if you feel like my balance are crap ( it is )
